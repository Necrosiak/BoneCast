## Что он делает

BoneCast помещает всё необходимое для стриминга на **Twitch** прямо в **меню быстрого доступа** Steam — без рабочего стола, без клавиатуры, полностью управляется геймпадом от входа до выхода в эфир.

- **Один вход в Twitch** *(код устройства — удобно с геймпадом)*: вы вводите короткий код на `twitch.tv/activate`, и BoneCast **получает ключ трансляции автоматически**. Больше никакого копирования-вставки.
- **Выход в эфир из QAM** — одна кнопка запускает и останавливает поток RTMP. Во время трансляции отображается значок **● В ЭФИРЕ**.
- **Редактируемый заголовок трансляции** — задаётся из плагина, изменяется **даже в эфире**.
- **Автоматическая категория игры** — считывается из запущенной игры Steam (работает и для ярлыков не из Steam) и обновляется на лету. Возвращается к *Just Chatting*, когда у игры нет подходящей категории на Twitch.
- **Прозрачный оверлей чата** — чат Twitch только для чтения, отрисованный **поверх игры** в игровом режиме (плоскость external-overlay gamescope), с нативными эмоциями + **BTTV / 7TV / FFZ**. Перемещаемый и настраиваемый.
- **Отключение микрофона в эфире** — отключите микрофон **в трансляции** одним касанием, не останавливая вещание (и не заглушая звонок в Discord).
- **Настройки трансляции для каждого аккаунта** — разрешение, кадры/с, битрейт видео/аудио, интервал ключевых кадров и **автоопределяемый кодировщик** (NVENC ▸ VAAPI ▸ программный x264). Предлагаются только кодировщики, которые действительно работают на вашем оборудовании.
- **Аудиомост Discord** *(когда установлен [Steamcord](https://github.com/Necrosiak/Steamcord))*: необязательный переключатель подмешивает ваш **голос из Discord** в трансляцию, чтобы вашу компанию слышали зрители — при этом вы продолжаете слышать их как обычно.

## Как это работает

У Twitch нет API для отправки видео, поэтому выход в эфир всегда означает **отправку RTMP** под капотом. BoneCast делает это за вас:

1. **Поток OAuth по коду устройства** входит в Twitch и получает ключ трансляции, заголовок и категорию через API Helix — вы никогда не касаетесь ключа.
2. Кадр игры захватывается из **gamescope** в устройство обратной петли `v4l2`, а **ffmpeg** кодирует его (аппаратно, если доступно, иначе программно `libx264`) и отправляет на `rtmp://…/<ваш-ключ>`.
3. **Оверлей чата** — это прозрачная поверхность WebKit, продвинутая на плоскость external-overlay gamescope, которая анонимно читает IRC Twitch и отображает эмоции.

Всё управляется из QAM и переживает перезагрузки.

## Установка

1. Установите [Decky Loader](https://github.com/SteamDeckHomebrew/decky-loader).
2. Возьмите последний `BoneCast.zip` со страницы [Releases](https://github.com/Necrosiak/BoneCast/releases) или установите его из магазина Decky, когда он станет доступен.
3. Откройте **меню быстрого доступа → BoneCast**, войдите в Twitch и выходите в эфир.

BoneCast **обновляется автоматически** из релизов GitHub (переключается в разделе *Обновления* плагина).
