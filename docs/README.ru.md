## Что он делает

BoneCast помещает всё необходимое для стриминга на **Twitch** прямо в **меню быстрого доступа** Steam — без рабочего стола, без клавиатуры, полностью под геймпад от входа в аккаунт до выхода в эфир.

- **Один вход в Twitch** *(код устройства — удобно с геймпада)*: вы вводите короткий код на `twitch.tv/activate`, и BoneCast получает ваш **ключ трансляции автоматически**. Больше никакого копирования вручную.
- **Выход в эфир из QAM** — одна кнопка запускает и останавливает RTMP-поток. Во время трансляции отображается значок **● В ЭФИРЕ**.
- **Редактируемое название трансляции** — задаётся из плагина, меняется **даже во время эфира**.
- **Автоматическая категория игры** — считывается из запущенной игры Steam (работает и для ярлыков не из Steam) и обновляется на лету. Если у игры нет подходящей категории Twitch, используется *Just Chatting*.
- **Прозрачный оверлей чата** — чат Twitch (только чтение), отрисованный **поверх игры** в игровом режиме (слой external-overlay gamescope), с нативными эмодзи + **BTTV / 7TV / FFZ**. Положение, размер и прозрачность настраиваются вживую из QAM, и оверлей остаётся видимым, даже когда Big Picture в фокусе.
- **🎬 Мгновенные клипы** — в эфире одна кнопка вырезает последние ~30 секунд через API Twitch; клип появляется на вашей панели управления примерно через 15 секунд.
- **💬 Пишите в свой собственный чат** — отправляйте сообщения в свой чат Twitch прямо из QAM, без акробатики с клавиатурой поверх рабочего стола.
- **⏸️ Режим BRB** — одно нажатие заменяет изображение игры аккуратным экраном паузы и отключает микрофон; ещё одно возвращает вас обратно. Трансляция никогда не прерывается.
- **⏺️ Локальная запись** — сохраняйте сессию в MKV в `~/Видео/BoneCast/`, параллельно с эфиром или **вообще без стриминга** (ключ трансляции не задействован).
- **Отключение микрофона в эфире** — отключите микрофон **в трансляции** одним нажатием, не останавливая эфир (и не заглушая свой звонок в Discord).
- **Настройки трансляции для каждого аккаунта** — разрешение, fps, битрейт видео/аудио, интервал ключевых кадров и **автоматически определяемый кодировщик** (NVENC ▸ VAAPI ▸ программный x264). Предлагаются только кодировщики, которые действительно работают на вашем железе.
- **Аудиомост Discord** *(если установлен [Steamcord](https://github.com/Necrosiak/Steamcord))*: опциональный переключатель подмешивает ваш **голосовой чат Discord** в трансляцию, чтобы зрители слышали вашу группу — а вы продолжаете слышать её как обычно.

## Как это работает

У Twitch нет API для отправки видео, поэтому выход в эфир всегда означает **RTMP-push** под капотом. BoneCast берёт это на себя:

1. **OAuth-поток с кодом устройства** входит в ваш аккаунт Twitch и получает ключ трансляции, название и категорию через API Helix — вы никогда не касаетесь ключа.
2. Изображение игры захватывается из **gamescope** в loopback-устройство `v4l2`, а **ffmpeg** кодирует его (аппаратно, если возможно, иначе программно через `libx264`) и отправляет на `rtmp://…/<ваш-ключ>`.
3. **Оверлей чата** — это прозрачная поверхность WebKit на слое external-overlay gamescope, которая анонимно читает IRC Twitch и отображает эмодзи.

Всё управляется из QAM и переживает перезагрузки.

## Установка

1. Установите [Decky Loader](https://github.com/SteamDeckHomebrew/decky-loader).
2. Включите **режим разработчика** в общих настройках Decky, затем настройки Decky → **Разработчик** → *Установить плагин по URL*:
   `https://github.com/Necrosiak/BoneCast/releases/latest/download/BoneCast.zip`
   (или возьмите `BoneCast.zip` со страницы [Releases](https://github.com/Necrosiak/BoneCast/releases) и установите из ZIP).
3. Откройте **меню быстрого доступа → BoneCast**, войдите в Twitch и выходите в эфир.

BoneCast **обновляется автоматически** через GitHub Releases (отключается в разделе *Обновления* плагина).

## 🐧 Совместимость

BoneCast нацелен на **все дистрибутивы Linux**, способные запускать Steam в
игровом режиме / Big Picture: одна сборка, определение всего внешнего во
время работы (ffmpeg, libx264, v4l2loopback, GStreamer) и точная команда
установки для вашего пакетного менеджера, показываемая в QAM, когда чего-то
не хватает. Заметки о пакетах по дистрибутивам: [OS-NOTES.md](OS-NOTES.md).
